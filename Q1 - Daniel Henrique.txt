HTML: O html serve como a estrutura ou esqueleto da página, é onde se posiciona os principais textos titulos, formulários, sua principal utilização como linguagem de marcação é estruturar a página Web em si,
CSS: Conhecida com linguagem de 'estilização' tem sua principal aplicação em dá cor estilo e personalidade a página, transformando textos e imagens soltas frutos do HTML, em textos com uma organização e distribuição na página de maneira mais visível e clara, utilizando desde modificações simples como o tipo de fonte à posição que uma texto ou imagem se posicionam na tela ou até como é a resposta de uma passage de mouse em um buttão, e muito mais isso tudo para poder proporcionar uma "exeperiência" para o usuário
JavaScript: O JavaScript por ser uma linguagem de programação, diferente dos outros dois, tem aplicação tanto no Front-end quanto no Back-End, seja no 'front' fornecendo uma página "dinamica" e animada dando assim novamente uma experiência melhor ao cliente como tbm alterando estruturas do CSS E HTML utilizando de Bibliotecas e Frameworks para esssa aplicação. O trabalho do JavaScript não acaba só ai, podemos ver o JavaScript sendo aplicado no Back-end tbm ou até mesmo consumindo API, podemos sitar por exemplo 2 frameworks para isso(Node.js e Next.js) uma das linguagens mais dinâmicas para a construção de páginas web, 
BONUS: O JavaScript pode ser utilizado no desenvolvimento Mobile tbm, com o uso do ReactNative
